# Deckplating User Guide

Hosted guide:

<https://deckplatingsetup.netlify.app/user-guide.html>

Use this guide to teach a new Religious Ministry Team how to use the app after setup is complete.

## Quick Start

1. Open the app from the phone home screen or browser.
2. Select your name.
3. Enter your 4-digit PIN.
4. Tap **Locate Me** on the **Check In** screen.
5. If a mapped location appears, select one or more units at that location and tap **Check In**.
6. If no location appears, tap **Manual unit lookup**, choose the location or one unmapped unit, then submit.
7. Review the confirmation screen.
8. Optional: check **Confidential care provided** or **Referral provided** if needed.
9. Tap **Done**.
10. Review **Coverage**, **Map**, and **Mission Board** as needed.

## App Philosophy

Deckplating is a lightweight coverage-awareness tool.

It is built to answer practical questions:

- Who has been visited recently?
- Which commands, departments, divisions, or tenant units need attention?
- Where are the mapped public locations?
- What meaningful coverage progress has the team made this month?

It is not a counseling record, case-management tool, official system of record, or place to store sensitive data.

## Safe Use

- Use Deckplating only for unclassified, non-sensitive coverage tracking.
- Do not enter CUI, classified information, counseling notes, medical details, home addresses, or sensitive operational locations.
- Map only publicly identifiable buildings or general areas.
- If a location should not be broadly shared, leave it unmapped and use manual check-in.

For the full policy, read [SAFE_USE.md](SAFE_USE.md).

## Screen-by-Screen Guide

### 1. Login

- Select your team roster name.
- Enter your 4-digit PIN.
- The app uses your name, PIN, and registered device to sign you in.

### 2. Check In

The Check In screen is the main working screen.

Normal flow:

- Tap **Locate Me**.
- Wait for the nearest mapped location to load.
- Select one or more units attached to that location.
- Tap **Check In**.

Manual flow:

- Tap **Manual unit lookup** if GPS is unavailable or the unit is not mapped.
- Choose one mapped location and one or more attached units, or choose one unmapped unit.
- Tap **Submit Manual Check-In**.

Confirmation:

- Review checked-in units, date/time, and points.
- Use **Undo this check-in** if you made an immediate mistake.
- Optional indicators save automatically when checked.
- Tap **Done** when finished.

### 3. Coverage

Use **Coverage** to see what needs attention first.

- Filter by area, unit type, overdue, never visited, or date range.
- Tap a command/department card to open its detail panel directly under that card.
- Review recent check-ins, last visitor, status, and indicator badges.
- Use the **Reports** section to load referral/confidential-care totals by mapped location and date range.

### 4. Map

Use **Map** to review mapped public locations and the units attached to them.

- Online: shows the interactive map and location circles.
- Offline: shows the cached location list instead of map tiles.
- Tap a mapped location card to open its detail panel directly under that card.

### 5. Mission Board

Mission Board emphasizes meaningful coverage, not raw volume.

- Score remains the main ranking metric.
- Supporting metrics include meaningful visits, distinct units, recovered units, and active days.
- Badges reward recovery, breadth, consistency, and first visits.

### 6. Settings

Use **Settings** to:

- change identity,
- refresh your working context,
- review the Safe Use section.

You cannot switch identities while queued visits are waiting to upload.

### 7. Admin

Use **Admin** for local app setup and correction work.

Sections:

- **Locations**: create or edit mapped public locations, units, and team members.
- **Activity Log**: filter check-ins and correct mistakes without deleting history.
- **Settings**: set Mission Board tone.

## Common Workflows

### Daily deckplate round

1. Open the app.
2. Read the Mission Brief.
3. Tap **Locate Me**.
4. Select units at the current location.
5. Submit the check-in.
6. Add optional indicators only if appropriate.
7. Move to the next location.

### GPS does not find the location

1. Confirm phone location permission is set to precise/high accuracy.
2. Tap **Locate Me** again.
3. If needed, use **Manual unit lookup**.
4. Ask an admin to verify the mapped location, coordinates, and radius later.

### Offline use

1. Launch the app once online before relying on offline use.
2. When offline, the sync bar shows cached mode or pending uploads.
3. Submit check-ins normally.
4. Reopen the app or tap **Sync Now** once connectivity returns.

### Fixing a mistake

- Immediate user mistake: use **Undo this check-in** on the confirmation screen.
- Later correction: use **Admin > Activity Log**.

## What To Teach New Users First

Train new users on these five tasks before anything else:

1. Sign in with name and PIN.
2. Use **Locate Me**.
3. Use **Manual unit lookup**.
4. Submit a check-in and read the confirmation screen.
5. Recognize the sync bar states.

## Best Training Order

1. Quick Start
2. Check In
3. Coverage
4. Map
5. Mission Board
6. Settings
7. Admin for local leads only
