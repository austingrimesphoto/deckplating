Self-hosted RMT setup

Start here. Do these steps in order.

This page walks you through making your own Deckplate Coverage app. Keep this tab open. Open each button in a new tab, finish that task, then come back here for the next step.

Do not skip ahead. You need accounts first, then your app copy, then the database, then Netlify.

Step 1

Create or open your accounts

You need these three accounts before anything else will work. Use command-owned accounts if your local policy allows it.

Create GitHub Account Create Supabase Account Create Netlify Account Open GitHub Login
Save these logins somewhere approved. Do not make the whole setup depend on one person’s personal account.

Step 2

Make your own copy of the app

This gives your RMT its own copy of the code. Keep your copy private unless your command intentionally wants it public.

Create My App Copy
If GitHub asks you to sign in, sign in and continue.

Step 3

Create your Supabase database

Open Supabase, create a new project, then run the schema/migrations block and the starter-data block below.

Open Supabase Projects

Step 4

Generate the secret settings for Netlify

Supabase gives you two values Netlify needs. Paste them here. This runs only in your browser and does not send values anywhere.

In Supabase, find these under Project Settings, then API. Use the service_role key, not the anon key.

Step 5

Deploy your app to Netlify

Netlify turns your GitHub copy into a live website. Import your GitHub repository, add the values from Step 4, then deploy.

Open Netlify Import

Step 6

Launch and finish setup

When Netlify says the deploy is published, open the site URL. Then finish your real local setup inside the app.

Step 7

Send pilot feedback back to the app owner

Use this form during or after the pilot. Submissions are stored by Netlify Forms on this setup site, so the app owner can review them without copy and paste.

Keep feedback unclassified and non-sensitive. Do not include counseling details, medical details, home addresses, personal identifying information, or sensitive operational locations.

Direct feedback link for pilots: https://deckplatingsetup.netlify.app/#feedback
